# 《重生之贵妇》5 分钟 Demo 游戏设计 · 基于 Act I

## 0. Demo 定位

- 品类建议: 乙女向 AVG + 恋爱悬疑 + 时空重生题材。
- 理由: Act I 的核心吸引力不是单纯甜宠，而是殷蕙带着前世记忆，在夫妻关系、王府规矩、娘家危机中“重新下注”。
- 【设计】表现形式建议: 2D 立绘 AVG 为主，关键节点插入 FMV 式镜头语言与动态 CG。
- 【设计】Demo 目标: 5 分钟内让玩家理解“她死过一次/悔过一次/这一次要改命”。

### 目标用户画像

- 喜欢重生、宅斗、贵女成长、先婚后爱题材的女性向玩家。
- 喜欢“冷面夫君逐渐在意”“女主清醒不恋爱脑”的乙女/言情用户。
- 喜欢 5-10 分钟高密度剧情体验、愿意为美术和情绪氛围付费的互动叙事用户。

### 核心玩法一句话

- 【设计】玩家扮演重生后的殷蕙，在前世记忆提示下，于“隐忍、试探、改局”之间选择行动，改变王府与殷家的命运节点。

### 与正式版的关系

- Demo 覆盖原文: SC-001 雨夜纳妾、第1章；SC-002 梦醒十年、第2-3章；SC-003 三门送别、第3-4章；SC-006 归来入帐、第13-14章的情感钩子作为结尾预告。
- Demo 不完整覆盖 Act I，只截取“纳妾崩塌 → 重生确认 → 王府规矩教学 → 夫妻关系新悬念”。
- 正式版承接点: 从咸宁八年七月清晨，魏曕进京前的王府送别与殷蕙第一次请准回娘家开始。
- 【设计】Demo 结尾存档落点: “重生记忆库”解锁，显示三条未来危机: 温如月、祖父横死、冯腾重伤。

## 1. 素材选择理由

### 选用场景

- 【选用】SC-001 雨夜纳妾: 一开场即给出“正妻被背叛、十年婚姻崩塌”的强钩子，台词“我不同意。”适合作为 Demo 第一记忆点。
- 【选用】SC-002 梦醒十年: 这是“重生贵妇”概念成立的核心场景，能让玩家在 1 分钟内理解时间倒流与改命目标。
- 【选用】SC-003 三门送别: 能低成本展示燕王府层级、规矩、群像压迫感，并提供第一处正式选项。
- 【选用】SC-006 归来入帐: 不完整展开，只作为结尾预告，展示魏曕对殷蕙变化的注意，制造乙女向期待。
- 【选用】EV-006 殷蕙回殷家与祖父重逢: 作为 Demo 中段闪回插针，提供情感高潮的亲情版本，引用“祖父在呢，阿蕙不哭。”。

### 弃用场景

- 【弃用】SC-004 主动结交: 信息价值高，但涉及魏楹、廖秋娘、廖十三、开铺多线，5 分钟内容量过大。
- 【弃用】SC-005 馍香入府: 烟火气强、适合正式版日常展示，但 Demo 首发钩子不如重生与纳妾直接。
- 【弃用】SC-007 冯腾避祸: “前世危机改写”玩法很强，但需要先解释护卫所、冯腾旧祸、龙阳谣言，理解成本偏高。
- 【弃用】SC-008 过继密谈: 家族线厚重，适合正式版第一章后中段展开；Demo 直接使用会削弱“重生开局”的清晰度。
- 【弃用】SC-009 神仙戏谑: 甜度与传播截图价值高，但缺少“重生贵妇”核心冲击。
- 【弃用】SC-010 寿春堂火: 情绪激烈、Act I 高潮强，但涉及性侵未遂、火灾、家法鞭笞、未成年女性受害，作为 5 分钟招商 Demo 风险过高。

### 为什么不选廖秋娘 climax（SC-010）

- SC-010 是 Act I 的收束高潮，不是开局概念展示。
- SC-010 需要玩家已理解殷闻、廖秋娘、廖十三、殷墉、殷家名声与报官困境，否则只剩刺激事件。
- SC-010 的敏感内容包括性侵未遂、药物迷晕、未成年女性受害、暴力鞭笞，平台合规与发行展示风险高。
- SC-010 会把 Demo 品类误导成复仇犯罪悬疑，而本作核心是“重生后在婚姻、家族、权力中改写命运”。
- 【设计】可在 Demo 结尾以“记忆库灰色节点”埋伏笔，不直接演出。

## 3. 关键场景卡（Demo 内详版）

### 【DSC-01｜雨夜纳妾】

- 原著对应: SC-001（第1章）
- 时间 / 地点 / 天气氛围: 景和二年四月雨夜；京城蜀王府；初夏小雨，室内压抑。
- 在场人物: 殷蕙（CH-01）、魏曕（CH-02）、温如月（CH-03）、金盏（CH-04）。
- 玩家 POV: 殷蕙（CH-01）。
- 场景目标: 让玩家在 30 秒内理解殷蕙前世婚姻崩塌的原因。
- 冲突焦点: 正妻殷蕙不愿魏曕纳温如月为妾，而魏曕以丈夫与王爷身份冷处理。
- Beats:
  1. 雨声先入，温如月白衣跪影出现在竹风堂方向。
  2. 魏曕回府，安排温如月住近主宅的竹风堂。
  3. 夜里魏曕告知殷蕙，过段时日纳温如月为妾。
  4. 殷蕙第一次强硬反对。
  5. 魏曕用“睡吧，明早再说。”压下冲突。
  6. 画面随殷蕙情绪坠入黑雨，转入重生。
- 关键台词:
  1. “我不同意。”（殷蕙，第1章）
  2. “睡吧，明早再说。”（魏曕，第1章）
- 视觉意象:
  1. 雨网。
  2. 白裙梨花。
  3. 黑帐。
  4. 两床被子。
  5. 竹风堂。
- CG / 立绘 / 特效需求:
  1. 温如月白衣跪影半身 CG，脸部可虚化，重点表现“介入感”。
  2. 殷蕙雨夜单人立绘: 发髻微乱、眼眶泛红、手攥被角。
  3. 魏曕冷调半身立绘: 深色寝衣、表情克制。
  4. 雨水遮罩转场: 雨线从竖向变为墨线，吞没画面。
  5. 【设计】信任崩塌 UI: 屏幕中央短暂裂纹，提示“前世记忆锚点已记录”。
- 音乐提示:
  1. 开场无旋律，只用雨声和低频。
  2. 殷蕙说“我不同意。”时 BGM 突然抽空。
  3. 魏曕说“睡吧，明早再说。”后进入压抑弦乐。
- UI 提示:
  1. 需要弹出第一组选项。
  2. 不显示好感变化，避免玩家误以为能改变前世结局。
  3. 【设计】弹出“前世不可改写，本次选择将影响殷蕙的内心倾向”。
- 剧情价值:
  1. 建立重生触发情绪。
  2. 建立温如月阴影。
  3. 建立魏曕“行动冷硬、言语压制”的前世印象。
  4. 建立殷蕙从顺从到反抗的第一声。

### 【DSC-02｜梦醒十年】

- 原著对应: SC-002（第2-3章）
- 时间 / 地点 / 天气氛围: 咸宁八年七月初九子夜至清晨；燕王府澄心堂；灯火昏黄，梦境感。
- 在场人物: 殷蕙（CH-01）、魏曕（CH-02）、金盏（CH-04）、衡哥儿（CH-05）。
- 玩家 POV: 殷蕙（CH-01）。
- 场景目标: 确认殷蕙回到十年前，并把“重生”变成可游玩的记忆系统。
- 冲突焦点: 前世记忆与眼前现实冲突；殷蕙必须迅速判断自己该如何活第二次。
- Beats:
  1. 殷蕙从雨夜惊醒，发现紫檀苏绣屏风不该出现在此时。
  2. 她看见年轻无须的魏曕，意识到时间错位。
  3. 她向金盏确认日期。
  4. 金盏回答“今年是咸宁八年，今日是七月初九。”
  5. 殷蕙抱起四个月大的衡哥儿，情绪几乎崩溃。
  6. 【设计】玩家点击三枚记忆碎片，初识未来危机。
- 关键台词:
  1. “今年是咸宁八年，今日是七月初九。”（金盏，第2章）
  2. “三爷尽管放心，我会照顾好衡哥儿。”（殷蕙，第2章）
- 视觉意象:
  1. 年轻无须魏曕。
  2. 铜镜少女脸。
  3. 浴桶冷水。
  4. 婴儿月牙笑。
  5. 昏黄灯火。
- CG / 立绘 / 特效需求:
  1. 铜镜双影 CG: 镜中是十年前少女脸，镜外叠加前世雨夜残影。
  2. 衡哥儿襁褓 CG: 小手抓住殷蕙衣襟。
  3. 殷蕙“震惊 / 克制 / 落泪”三表情立绘。
  4. 年轻魏曕睡侧影立绘或动态剪影。
  5. 【设计】记忆碎片 UI: 温如月、殷墉、冯腾、廖秋娘只以墨色剪影出现。
- 音乐提示:
  1. 雨声转为耳鸣。
  2. 日期确认时加入铜漏滴水。
  3. 衡哥儿出现时弦乐转暖，但底部保留不安鼓点。
- UI 提示:
  1. 弹出“重生记忆库”首次引导。
  2. 可点击三枚碎片，但只显示关键词，不展开完整真相。
  3. 第二组选项出现: 今生第一念。
- 剧情价值:
  1. 完成重生设定说明。
  2. 建立衡哥儿作为情感锚点。
  3. 把 Act I 的四大修正目标做成系统雏形。
  4. 让玩家知道殷蕙不是单纯重活，而是带着任务重活。

### 【DSC-03｜三门送别】

- 原著对应: SC-003（第3-4章）
- 时间 / 地点 / 天气氛围: 咸宁八年七月清晨；燕王府侧殿、静好堂、西华门；晨光清澄但规矩森严。
- 在场人物: 殷蕙（CH-01）、魏曕（CH-02）、纪纤纤（CH-06）、徐清婉（CH-07）、燕王（CH-08）、温夫人（CH-09）、衡哥儿（CH-05）。
- 玩家 POV: 殷蕙（CH-01）。
- 场景目标: 展示燕王府权力秩序，并让玩家第一次体验“说话方式影响局势”。
- 冲突焦点: 纪纤纤奚落殷蕙，引出二郎哭闹与燕王惩戒二房；殷蕙必须在王府规矩里保护自己。
- Beats:
  1. 殷蕙抱衡哥儿随众送魏曕进京。
  2. 徐清婉注意到殷蕙从容与美貌。
  3. 纪纤纤借京城话题奚落殷蕙。
  4. 二郎被激哭闹，要去京城。
  5. 燕王出场查问。
  6. 二郎撒谎被揭破，二房被禁足至中秋。
  7. 魏曕安抚殷蕙“你并未犯错，不必担心什么。”
- 关键台词:
  1. “一大早的，谁在哭闹？”（燕王，第3章）
  2. “你并未犯错，不必担心什么。”（魏曕，第4章）
- 视觉意象:
  1. 三重宫门。
  2. 晨光长影。
  3. 侧殿肃立。
  4. 婴儿襁褓。
  5. 王府朱门。
- CG / 立绘 / 特效需求:
  1. 王府三重门远景，强调“进出内城，便要过三重门”的规训感。
  2. 纪纤纤明艳挑衅立绘。
  3. 徐清婉端庄观察立绘。
  4. 燕王低机位威严立绘，背景压暗。
  5. 魏曕侧身回头 CG: 晨光中看向殷蕙母子。
- 音乐提示:
  1. 群像入场用礼乐低配版。
  2. 二郎哭闹时加入急促木鱼点。
  3. 燕王出场时全部人声压低，只留靴声与低鼓。
- UI 提示:
  1. 第三组选项出现: 面对纪纤纤如何回应。
  2. 【设计】王府规矩条首次出现，不叫“好感”，叫“失仪风险”。
  3. 选项后显示短提示: “守规矩，不等于低头。”
- 剧情价值:
  1. 快速展示后宅不是单纯雌竞，而是权力秩序。
  2. 让玩家理解殷蕙的策略: 不主动生事，但会借规矩自保。
  3. 初步修正玩家对魏曕的单一负面印象。

## 4. 首屏 5 秒钩子设计

### 候选方案 A：雨夜一句“我不同意。”

- 视觉钩子: 黑屏后骤亮，雨线打在窗纸上，温如月白衣跪影一闪。
- 听觉钩子: 暴雨、烛火噼啪、殷蕙压抑到发颤的一句“我不同意。”
- 台词钩子: “我不同意。”（殷蕙，第1章）
- 悬念钩子: 她不同意什么？为什么一个贵妇会在雨夜失控？
- 利: 原文高光最强，最短时间建立女性压迫与反抗。
- 弊: 若没有后续快速重生，可能被误判为传统宅斗苦情。
- 适合: 发行展示、短视频投放、试玩首屏。

### 候选方案 B：铜镜中少女脸

- 视觉钩子: 铜镜里先映出二十五岁疲惫眼神，再闪成十六七岁少女脸。
- 听觉钩子: 铜漏一滴，心跳骤停。
- 台词钩子: “今年是咸宁八年，今日是七月初九。”（金盏，第2章）
- 悬念钩子: 她为什么突然回到十年前？
- 利: “重生”概念直给，题材识别清晰。
- 弊: 情绪爆点不如雨夜纳妾，首屏冲击稍弱。
- 适合: 商店页视频第二版，面向重生题材核心用户。

### 候选方案 C：婴儿小手抓住衣襟

- 视觉钩子: 昏黄灯下，四个月大的衡哥儿小手抓住殷蕙衣襟。
- 听觉钩子: 婴儿笑声与雨夜哭声叠化。
- 台词钩子: “三爷尽管放心，我会照顾好衡哥儿。”（殷蕙，第2章）
- 悬念钩子: 这个孩子前世发生了什么，让她如此失态？
- 利: 情感柔软，能吸引亲情向用户。
- 弊: 首屏不够锋利，可能慢热。
- 适合: 正式版章节内情绪回收，不建议做 Demo 首屏。

### 最终推荐

- 推荐候选方案 A。
- 理由:
  1. “我不同意。”是 Act I 最短、最有传播力的反抗台词。
  2. 雨夜、白衣、纳妾、正妻四个视觉信息可在 5 秒内完成理解。
  3. 紧接 SC-002 梦醒十年，可迅速从“苦情”转为“改命爽感”。
  4. 对发行/投资人而言，开场识别度最高。

## 5. 交互选项设计

### 选项 1：雨夜回应

- 选项时刻: 0:25-0:45，SC-001，魏曕提出纳温如月为妾后。
- 玩家问题: “他要纳她为妾，你要如何开口？”
- 选项 A: “忍下去，问他可曾想过我。”
- 选项 B: “直视他，说: 我不同意。”
- 选项 C: “冷笑，问她要住进哪一处院子。”
- 各选项后果:
  1. 立即表现 A: 殷蕙声音发颤，魏曕仍冷处理。
  2. 立即表现 B: 触发原文台词“我不同意。”，屏幕裂纹最强。
  3. 立即表现 C: 殷蕙语气更冷，温如月白衣剪影闪回。
  4. 延时表现 A: 重生后“隐忍值”初始 +1，内心独白更痛。
  5. 延时表现 B: 重生后“清醒值”初始 +1，记忆库标题更锋利。
  6. 延时表现 C: 重生后“防备值”初始 +1，魏曕观察她时提示更早出现。
- 【设计】此选项想让玩家感受到:
  1. 代入殷蕙十年婚姻被否定的窒息。
  2. 明白前世无法改写，但“今生如何成为她”可以选择。
  3. 第一次感受到“女主不是任人摆布”。

### 选项 2：今生第一念

- 选项时刻: 2:35-2:55，SC-002，殷蕙抱起衡哥儿并确认重生后。
- 玩家问题: “这一世，你最先要守住什么？”
- 选项 A: “衡哥儿。”
- 选项 B: “祖父。”
- 选项 C: “我自己。”
- 各选项后果:
  1. 立即表现 A: 衡哥儿小手 CG 停留更久，触发母子心跳音。
  2. 立即表现 B: 闪回殷墉台词“祖父在呢，阿蕙不哭。”（第7章）。
  3. 立即表现 C: 铜镜中少女脸变清晰，殷蕙擦去眼泪。
  4. 延时表现 A: 正式版新手任务优先提示“衡哥儿教养边界”。
  5. 延时表现 B: 正式版新手任务优先提示“回殷家探亲”。
  6. 延时表现 C: 正式版新手任务优先提示“换回明艳装扮，请准探亲”。
- 【设计】此选项想让玩家感受到:
  1. 殷蕙的改命不是恋爱唯一目标。
  2. 母子、祖孙、自我三条动机并行。
  3. 玩家可以决定 Demo 结束后正式版的情绪入口。

### 选项 3：面对纪纤纤奚落

- 选项时刻: 3:45-4:10，SC-003，纪纤纤借京城话题奚落殷蕙后。
- 玩家问题: “王府众目睽睽，你要怎样回？”
- 选项 A: “低头认下，不争一时。”
- 选项 B: “顺着规矩，把话轻轻拨回去。”
- 选项 C: “反问二嫂，京城规矩是否教人当众刺人。”
- 各选项后果:
  1. 立即表现 A: 殷蕙失仪风险不升，但纪纤纤气焰更足。
  2. 立即表现 B: 纪纤纤一时无话，徐清婉看殷蕙一眼。
  3. 立即表现 C: 群像私语变强，失仪风险上升。
  4. 延时表现 A: 魏曕安抚时语气更平。
  5. 延时表现 B: 魏曕安抚时出现“他似乎注意到你的分寸”。
  6. 延时表现 C: 燕王出场压迫感更强，但二房受罚结果不变。
- 【设计】此选项想让玩家感受到:
  1. 王府不是想怼就怼的爽文空间。
  2. 殷蕙的厉害在于懂规矩、借规矩，而不是破坏规矩。
  3. “守规矩即可自在”是 Act I 的核心策略之一。

## 6. 结尾 Cliffhanger 设计

### 候选 A：悬念句

- 悬念句: “怎么没等我？”（魏曕，第14章）
- 呈现方式: 魏曕推门入澄心堂，看见殷蕙并未如前世那样等他。
- 评价: 乙女向最有效，短句里有关系变化、占有感、困惑感。

### 候选 B：视觉/场景切换

- 视觉/场景切换: 西华门三重门关闭，门缝中切入殷宅书房的祖父背影。
- 呈现方式: 屏幕浮现“祖父在呢，阿蕙不哭。”（第7章）后切黑。
- 评价: 亲情冲击强，适合突出殷家线，但恋爱向期待略弱。

### 候选 C：关键人物登场

- 关键人物登场: 温如月白衣剪影再次出现，记忆库标注“温如月: 此时已嫁绍兴？”。
- 呈现方式: SC-002 后的记忆碎片中，温如月节点短暂亮起。
- 评价: 悬疑强，但 Act I 只确认温如月此时已嫁绍兴，不能过度暗示她已在今生入局。

### 最终推荐

- 推荐候选 A。
- 理由:
  1. 与开场魏曕冷处理形成前后对照。
  2. 准确来自 SC-006 第14章，不虚构。
  3. 既能服务乙女用户，又能表达女主变化。
  4. 结尾不靠敏感事件，也能留下“他会怎么追问”的期待。

## 11. 潜在风险与替代方案

### 敏感 / 分级 / 平台合规风险

- 纳妾压迫:
  1. Demo 必然出现 SC-001 纳妾冲突。
  2. 表现重点放在情绪背叛与制度压迫，不渲染私密强迫。
  3. 魏曕“睡吧，明早再说。”保留为冷处理台词，但镜头避免身体压迫演出。
- 婚内亲密:
  1. SC-006 只做预告，不演出亲密过程。
  2. 双人对视保持暧昧距离，避免误导分级。
- 未成年女性受害:
  1. SC-010 不进入 Demo 正片。
  2. 廖秋娘只可在记忆库以灰色节点出现，不出现暴力画面。
- 家法鞭笞:
  1. 不作为 Demo 展示内容。
  2. 正式版如制作，建议以鞭声、人物反应和后果替代直接血腥镜头。
- 幼童受罚:
  1. SC-003 二郎哭闹与二房禁足可保留。
  2. 不表现体罚，只表现燕王威压与规矩结果。

### 5 分钟内容量 vs 玩家理解成本

- 当前方案信息量:
  1. 前世纳妾。
  2. 重生确认。
  3. 衡哥儿情感锚点。
  4. 王府规矩。
  5. 魏曕关系悬念。
- 风险:
  1. 角色较多，纪纤纤、徐清婉、燕王、温夫人同场可能让新玩家记不住。
  2. 记忆碎片如果展示过多，会打断情绪。
- 替代精简方案:
  1. 删除徐清婉与温夫人正脸立绘，仅以群像剪影出现。
  2. 侧殿只保留纪纤纤、燕王、魏曕与殷蕙的直接信息。
  3. 记忆库只显示三枚碎片: 温如月、祖父、衡哥儿。
  4. 将冯腾、廖秋娘、殷阆延后到正式版第一章后解锁。

### 品类误判风险

- 可能误判 1: 玩家以为是纯宅斗。
  - 解决: 开场 90 秒必须完成“重生确认”，不能长时间停在纳妾苦情。
- 可能误判 2: 玩家以为是纯乙女恋爱。
  - 解决: 第二选项加入“衡哥儿 / 祖父 / 我自己”，强调多线改命。
- 可能误判 3: 发行认为互动不足。
  - 解决: 5 分钟内放 3 个选项，至少 1 个影响正式版导入提示。
- 可能误判 4: 美术成本过高。
  - 解决: 温如月、王府群像先用剪影与半身立绘，CG 集中在 4-5 张高价值瞬间。
- 备选品类:
  1. 女性向重生权谋 AVG。
  2. 古风互动短剧 / FMV 式竖屏叙事。
  3. 轻经营 + 关系养成的古风剧情手游。
  4. 章节制付费视觉小说。
